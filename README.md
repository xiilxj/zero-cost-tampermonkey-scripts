# ⚡ Zero Cost Tampermonkey Scripts - 全能网页限制解除与媒体一键提取助手

![Banner](banner.png)

一个完全免费、零成本的油猴脚本项目。旨在解决日常网页浏览中的高频痛点，并提供无缝的无水印媒体资产抓取与被动收益打赏通道。

## 🌟 核心特性

- **🔓 全能网页限制解除**：自动破解任意网页的防复制、防剪切、防右键和防选中文本限制（支持各大网课、文档与新闻网）。
- **🖼️ 网页媒体一键抓取**：右下角提供极简浮动工具箱，点击“提取”即可一键抓取并预览全网页无水印图片，支持批量或单张保存。
- **🎨 极简现代 UI 体验**：使用精美的 Glassmorphism（玻璃拟物化）微动视效菜单，轻量级无感加载。
- **☕ 创作者赞助集成**：预置了优雅的“爱发电”赞助通道，轻松获取用户的被动打赏收入。

---

## 🚀 一键安装与使用

1. **第一步：安装脚本管理器**  
   确保你的浏览器已安装 [Tampermonkey（油猴）](https://www.tampermonkey.net/) 插件。

2. **第二步：一键激活脚本**  
   点击访问我们的 GreasyFork 官方发布页面：  
   👉 [GreasyFork 脚本主页](https://greasyfork.org/zh-CN/scripts/583996-%E5%85%A8%E8%83%BD%E7%BD%91%E9%A1%B5%E9%99%90%E5%88%B6%E8%A7%A3%E9%99%A4%E4%B8%8E%E5%AA%92%E4%BD%93%E4%B8%80%E9%94%AE%E6%8F%90%E5%8F%96%E5%8A%A9%E6%89%8B)  
   点击 **“安装此脚本”** 绿色按钮即可。

3. **第三步：开始使用**  
   安装后，刷新任意网页即可自动解除所有复制限制。右下角的 **🔓 小按钮** 可以展开媒体提取与赞助工具箱。
